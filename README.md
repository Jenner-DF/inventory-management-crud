# Inventory Management Web App

A simple CRUD inventory management web application built with **Next.js**, **TypeScript**, **Prisma**, **Neon (PostgreSQL)**, **shadcn/ui**, and **Zod**.  
You can add, view, edit, and delete products. .

---

## Features

- **Create** new products
- **Read** all products in a table
- **Update** product details
- **Delete** products (single or multiple)
- Form validation using **Zod**
- Responsive UI components using **shadcn/ui**
- Real-time state updates with **React Query**

---

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router / React 18)
- [TypeScript](https://www.typescriptlang.org/)
- [Prisma](https://www.prisma.io/) ORM
- [Neon](https://neon.tech/) PostgreSQL database
- [shadcn/ui](https://ui.shadcn.com/) components
- [Zod](https://zod.dev/) for form validation
- [React Hook Form](https://react-hook-form.com/) for forms
- [TanStack Table](https://tanstack.com/table/v8) for product tables
